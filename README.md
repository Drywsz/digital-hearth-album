# Digital Hearth Album

Este projeto é um diário digital artesanal, uma jornada delicada de admiração, valores e pequenas aventuras. Desenvolvido com uma stack moderna, ele explora conceitos de um "pequeno mundo feito para você", com seções interativas e uma experiência de usuário envolvente.

## 🚀 Tecnologias Utilizadas

*   **TypeScript:** Linguagem de programação para tipagem estática e escalabilidade.
*   **React:** Biblioteca JavaScript para construção de interfaces de usuário.
*   **Vite:** Ferramenta de build rápida para desenvolvimento web.
*   **Tailwind CSS:** Framework CSS utilitário para estilização rápida e responsiva.
*   **Radix UI:** Biblioteca de componentes UI acessíveis e personalizáveis.
*   **TanStack Router:** Roteamento de última geração para aplicações React.

## ✨ Funcionalidades

*   Navegação interativa com seções temáticas (Início, Mundo, Quem você é, Admiro, Cores, Aventuras, Fé, Caça, Sonhos, Carta, Final).
*   Design responsivo e moderno.
*   Experiência de usuário focada em narrativa e personalização.

## ⚙️ Como Rodar Localmente

1.  Clone este repositório:
    ```bash
    git clone https://github.com/Drywsz/digital-hearth-album.git
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd digital-hearth-album
    ```
3.  Instale as dependências:
    ```bash
    bun install # ou npm install / yarn install
    ```
4.  Inicie o servidor de desenvolvimento:
    ```bash
    bun dev # ou npm run dev / yarn dev
    ```
5.  Abra seu navegador e acesse `http://localhost:5173` (ou a porta indicada).
